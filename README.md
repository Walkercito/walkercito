## 🚀 What Gets Me Out of Bed

I'm completely **self-taught** and honestly, I love the journey more than the destination. Here's what keeps me coding:

- 💡 **Curiosity-driven development** – If it's new and shiny, I'm probably already breaking it

- 💾 **Open source advocate** – Like my **CopyClip** project that's helping Linux users everywhere
